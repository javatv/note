# 删除排序链表中的重复元素
⭐：简单

## 题目描述

[删除排序链表中的重复元素](https://leetcode-cn.com/problems/remove-duplicates-from-sorted-list/)

给定一个已排序的链表的头 `head` ， 删除所有重复的元素，使每个元素只出现一次。返回已排序的链表 。

**示例 1**：

```
输入：head = [1,1,2]
输出：[1,2]
```

**示例 2**：

```
输入：head = [1,1,2,3,3]
输出：[1,2,3]
```
